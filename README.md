# Autonomyx Partners

**Open Source Commercial Partner Delivery Network**

Autonomyx is building a certified commercial delivery network for open source agentic infrastructure, decision intelligence, and governed automation.

This repository defines the partner model, delivery standards, certification structure, marketplace rules, governance requirements, and contribution process for the Autonomyx partner ecosystem.

## Positioning

Autonomyx is not open core.

Autonomyx is an **Open Source Commercial Core** company:

- The software is open source.
- The business is commercial.
- The delivery is partner-led.
- The trust layer is governed.
- Customers pay for outcomes, implementation, operations, assurance, and lifecycle ownership.

Autonomyx does not sell a weak community edition and a hidden enterprise edition. Everyone gets the real platform. Commercial value comes from trusted delivery, governance, operations, compliance, industry packaging, certification, and managed lifecycle services.

## Getting started

1. Read `docs/partner-model.md`.
2. Choose the right program under `programs/`.
3. Create a partner profile using `templates/partner-profile.yaml`.
4. Submit the profile under `registry/partners/`.
5. Provide delivery, certification, or solution evidence as required.
